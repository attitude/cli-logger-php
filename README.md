# CLI Logger
A tiny PSR-3 Logger for CLI use

## Installation
```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/attitude/cli-logger-php"
        }
    ],
    "require": {
        "attitude/cli-logger": "dev-main"
    }
}
```
